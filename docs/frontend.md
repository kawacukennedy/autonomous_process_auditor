# Frontend Documentation

## Overview
The frontend is built with React, TypeScript, Vite, and Tailwind CSS.

## Pages
- Dashboard: KPIs and audit feed
- Process Configuration: Connector management
- Input: Log upload and analysis
- Findings: Results and remediation
- Agent Console: Trace visualization
- Auth: User login
- Reports: Report generation

## Components
- Header: Navigation
- Footer: Copyright
- KPICard: Metric display
- AuditFeed: Event stream
- Loading: Spinner
- Error: Error messages
- Modal: Dialogs

## Running
```bash
cd frontend
npm install
npm run dev
```
