# Autonomous Process Auditor (APA)

Continuously audit, detect, and autonomously remediate workflow inefficiency, compliance drift, and bottlenecks across enterprise systems using IBM watsonx Orchestrate.

## Architecture

- **Frontend**: React (TypeScript) + Tailwind CSS + Vite
- **Backend**: Node.js (Fastify) with TypeScript
- **Database**: MongoDB Atlas
- **AI/ML**: IBM watsonx.ai Granite models
- **Orchestration**: IBM watsonx Orchestrate

## Getting Started

1. Clone the repo
2. Set up Frontend: `cd frontend && npm install`
3. Set up Backend: `cd backend && npm install`
4. Run Frontend: `cd frontend && npm run dev`
5. Run Backend: `cd backend && npm run dev`
